Implementace

Pro implemetaci byl použit php framework Nette, a pro klientskou část další podpůrné knihovny/frameworky: bootstrap, jQuery, fonta-awesome.

Naše vlastní zdrojové kódy se tedy nazházejí jen v podadresářích www/ a app/

Případy užití pro roli Administratora jsou implementaované v souboru /app/presenters/AdminPresenter.php a v odpovídajících šablonách v adresáři /app/presenters/templates/Admin/

Případy užití pro roli Chovatele jsou implementaované v souboru /app/presenters/KeeperPresenter.php a v odpovídajících šablonách v adresáři /app/presenters/templates/Keeper/

Případy užití pro roli Klienta jsou implementaované v souboru /app/presenters/ClientPresenter.php a v odpovídajících šablonách v adresáři /app/presenters/templates/Client/

Instalace

  • rozbalit zip archiv standartním způsobem a přesunounout obsah adresáře chovna_stanice na patřičné umístnění dle pužitého HTTP serveru
  • nastavit root directory HTTP serveru na chovna_stanice/www/
  • vytvořit soubor chovna_stanice/app/config/config.local.neon s obsahem: database: dsn: 'mysql:host=localhost;dbname=chovna_stanice' user: chovna_stanice password: chovna_stanice
  • vytvořit MySQL databázi chovna_stanice a k ní uživatele chovna_stanice s heslem chovna_stanice
  • spustit/importovat soubor chovna_stanice/sql/db.sql

Požadavky

  • PHP verze 5.3.1 nebo vyšší
  • HTTP server s podporou .htaccess a mod_rewrite (Apache)
  • Další vyplývající z použitého frameworku, vizte: https://doc.nette.org/cs/2.3/requirements
